# Asteroids
This project is a space-shooter, similar to the arcade game Asteroids.

Recently my laptop was damaged and so I was unable to code. But my boredom got the best of me and I ended up programming on my smartphone.

Therefore, this project allowed me to learn deeply about touch controls with hands-on experience.

While the project is just a simple game, my main objective was to create some basic classes which I will be using in my future projects.
As such, this project can be thought of as a prototype for my future projects.

The major benefits I got from working on this project are:

• A Basic Object class: It is used for the ship, asteroids as well as the bullets.

• On-screen Joystick and Button: I used the joystick to give the player 360° of movement, while the button is used for shooting.

• Infinte Background: It allowed me to generate an endless space for the player to explore.

• Various Performance Techniques: During development, I came across various performance issues with some causing the game to be completely still. I created many solutions on my own as well as found a few solutions on online forums like StackOverflow and Reddit.

This is definitely one of the few projects which I enjoyed making to the fullest, albeit there were some times where it didn't feel like that.

At the end of the day, the tools and knowledge I acquired from this project will be beneficial for my future projects.
